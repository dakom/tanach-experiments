Generic Notes:

You must be using a modern web browser which supports WebGL (most updated versions do - unless you're on internet explorer).

If your want to adjust the resolution or get rid of scrollbars, you should usually resize your window and hit refresh

Where flipping is available, the initial settings are flipped horizontal, e.g. to simulate right-to-left

Source code is available here

Specific Notes:

Letters Palette

Keys:

Touch:  Get the location in Tanach
Spacebar:  Pause/Play animation
Up:  Isolate next letter
Down:  Isolate previous letter
1:  Regular
2:  Atbash
3:  Albam
F:  Flip Horizontal

Notes:

This experiment colors a pixel for every letter in Tanach

You can isolate any particular letter, as well as substitute all the letters according to atbash or albam

The color palette begins with random assignments, and animates with random speeds. You can pause it at any time To adjust the resolution, see above

Soft's are distinct (i.e. after tav is mem sofit, nun sofit, etc.) If you want to see the current chosen letter - look in the developer's console

Word Hilight
Keys:

Touch:  Get the location in Tanach
Buttons:  Change colors and speed
Edit Value:  Manually set current value
F:  Flip Horizontal

Notes:

This experiment highlights word matches throughout all of Tanach

It cycles from the first to last word in Tanach, and will pick up from where it last left off when paused/played

Note that currently it only searches per-word, not per-phrase

Gematria Hilight
Keys:

Touch:  Get the location in Tanach
Buttons:  Change colors and speed
Edit Value:  Manually set current value
F:  Flip Horizontal

Notes:

This experiment highlights gematria matches throughout all of Tanach, starting from lowest to highest and animating through

Unlike in the Letters Palette, Sofits are not distinct here (tav-400 is the highest)

Note that currently it only searches per-word, not per-phrase

Ratio Draw

Keys/Controls: None

Notes:

Currently set to Sofit-Atbash (gematria of each letter, transposed in atbash, and including the extra value for Sofit's)

Demo pages for Tanach Experiments
by David Komer

You must be using a modern web browser which supports WebGL (most updated versions do - unless you're on internet explorer).

If your want to adjust the resolution or get rid of scrollbars, you should usually resize your window and hit refresh

Gematria Hilight - Hilight matches in Tanach
Keys:

Spacebar:  Pause/Play animation
F:  Flip Horizontal
Edit Value:  Manually set current value

Notes:

This experiment shows a greyscale for every letter in Tanach and highlights matches to gematria

In animation mode, it automatically cycles from the lowest to highest value

Unlike in the Letters Palette below, Sofits are not distinct here (tav-400 is the highest)

It begins flipped horizontal, e.g. to simulate right-to-left, but you can flip that back if you want.

Letters Palette - view all the letters of Tanach as colors
Keys:

Spacebar:  Pause/Play animation
Up:  Isolate next letter
Down:  Isolate previous letter
1:  Regular
2:  Atbash
3:  Albam
F:  Flip Horizontal

Notes:

This experiment colors a pixel for every letter in Tanach

You can isolate any particular letter, as well as substitute all the letters according to atbash or albam

The color palette begins with random assignments, and animates with random speeds. You can pause it at any time

When you first start, the app will take the width of your screen and construct the pixels to fill it until the target height. To adjust the resolution, see above

Soft's are distinct (i.e. after tav is mem sofit, nun sofit, etc.) If you want to see the current chosen letter - look in the developer's console

It begins flipped horizontal, e.g. to simulate right-to-left, but you can flip that back if you want.


Ratio Draw currently set to Sofit-Atbash (gematria of each letter, transposed in atbash, and including the extra value for Sofit's)
